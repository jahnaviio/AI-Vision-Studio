# 👁️ AI Vision Studio

A real-time Computer Vision application that performs **Face Detection, Emotion Recognition, Gender Prediction, and Object Detection** using a live webcam feed. Built with **OpenCV**, **DeepFace**, and **YOLOv8**, and deployed with **Gradio** on **Hugging Face Spaces**.

## 🚀 Features

- 🎯 Real-time Face Detection
- 😊 Emotion Recognition
- 👤 Gender Prediction
- 📦 Object Detection with YOLOv8
- 📹 Live Webcam Processing
- 🌐 Interactive Web Interface using Gradio
- ☁️ Deployable on Hugging Face Spaces

## 🛠️ Tech Stack

- Python
- OpenCV
- DeepFace
- YOLOv8 (Ultralytics)
- Gradio
- Hugging Face Spaces
- GitHub

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/AI-Vision-Studio.git
cd AI-Vision-Studio
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
